Business Sales Data Analysis

This project analyzes a small business sales dataset to predict profit and customer spending trends using Python.


Steps:

1) Data Cleaning: Removed missing values and ensured correct data types.
2) Data Visualization: Created graphs to explore sales, profit, and customer behavior.
3) Prediction: Used linear regression to forecast profit based on future trends.



Tools Used:

- Python (Pandas, Matplotlib, Scikit-learn)
- Jupyter Notebook
- GitHub



Results:

The model provides a simple prediction of business profit and spending trends.
